Instructions

Memory Matrix is a game where you have to use your memory to fill the matrix and reproduce a pattern without mistakes.
  • the order in which you click the cells is not important.
  • in order to reproduce the pattern you must click only on the cells that belong to the pattern.
  • for every correct cell clicked you get 10 points
  • You will only get the chances to click the tile equals to the number of tiles hidden

    For Example:-
    If you have 7 tiles hidden only 7 Times you can click the tiles. If you clicked wrong tile 2 time Then you will get 20 Points less than the maximum point you can get from that level
